# 📚 StudyHub Planner

StudyHub Planner is a Django web application designed to help students organize their study tasks and assignments in a simple and efficient way.

---

## 🚀 Features

- User registration and authentication
- Login and logout system
- Create study assignments
- View personal assignments
- Mark assignments as completed or pending
- Delete assignments
- Each user has private data (secure system)

---

## 🛠️ Built With

- Python
- Django
- HTML
- Basic CSS (inline styling)

---

## 📂 Project Structure

- models.py → Database models (User, Assignment, Subject)
- views.py → Application logic
- urls.py → Routing system
- templates/ → Frontend pages
